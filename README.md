# Zahidul Islam Portfolio

Simple static portfolio site hosted with GitHub Pages.

## Pages

- `/` -> Main portfolio homepage
- `/bala/` -> Bala project landing page

## Run Locally

From the repo root:

```bash
python3 -m http.server 8080
```

Then open:

- `http://localhost:8080/`
- `http://localhost:8080/bala/`

## Deploy

Push to the `main` branch of the `islamz1214.github.io` repository.
GitHub Pages will publish automatically.
